# Spreadsheets

Use this skill for XLSX, CSV, TSV, and sheet-like business workflows: analysis, cleanup, modeling, QA, summaries, and stakeholder-ready tables.

## Workflow

1. Identify the workbook purpose, sheets, key columns, formulas, date ranges, currencies, units, and expected output.
2. Check data quality before analysis: missing values, duplicate rows, inconsistent labels, outliers, hidden assumptions, and stale time periods.
3. Preserve formulas and source data unless the user asks for destructive cleanup. Prefer adding derived columns, summary tabs, or clearly named outputs.
4. For modeling, state assumptions, inputs, outputs, sensitivities, and where manual updates are expected.
5. For review, verify totals, filters, joins, formulas, references, and formatting that could change interpretation.

## Delivery

Return a concise analysis, cleaned table, formula guidance, model structure, or updated spreadsheet artifact. Include verification checks and any unresolved data issues.
