# Word Docs

Use this skill for DOCX-style business writing and review: memos, proposals, briefs, policies, reports, and stakeholder-ready drafts.

## Workflow

1. Clarify the audience, purpose, delivery format, and any required sections.
2. Preserve the user's existing structure and voice unless the request asks for a rewrite.
3. Make edits concrete: headings, tracked issues, inline replacement text, executive summary, decisions, risks, and next steps.
4. For review work, separate blocking issues from polish and call out missing evidence, undefined terms, inconsistent claims, and approval-sensitive language.
5. For conversion work, preserve hierarchy, tables, footnotes, citations, and change history where the source format makes that possible.

## Delivery

Return a clean document outline or edited text that can be pasted into Word, or update the workspace file directly when file editing is part of the task. Include unresolved assumptions and verification notes.
